# Olympics-medal-analysis-excel
## Overview
This project analyzes historical Olympic medal data using Microsoft Excel to discover trends in medal distribution, dominant countries, and sport-wise performance over time.

## Key Insights
- The USA leads the overall medal tally across decades
- Certain countries have a higher gold-to-total medal ratio
- Specific sports contribute heavily to some nations' success
- Medal counts have increased over time with more events

## Tools Used
- *Microsoft Excel*
  - Pivot Tables
  - Conditional Formatting
  - Charts (Bar, Line, Pie)
  - Sorting, Filtering
